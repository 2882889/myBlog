####原因
“位或”和”位与”运算
“位或”: 0 | 0 = 0 ; 0 | 1 = 1 ; 1 | 1 = 1;
“位与”: 0 & 0 = 0 ; 0 & 1 = 0 ; 1 & 1 = 1;

对移位枚举进行”位或”运算
