# Mac软件--mos

## 前言

Mac黑苹果使用鼠标的时候，鼠标滚轮不是特别灵敏，经常向下滚动的时候，会出现页面向上的问题，使用这个`mos`软件可以完美解决这个问题

<!-- more -->
### 安装方法

#### 软件源码地址

[mos地址](https://github.com/Caldis/Mos)

#### 安装

```
brew cask install mos
```

命令会自动安装软件到`/Applications/Mos.app`

#### 更新

```
brew update
brew cask reinstall mos
```

更新后重新启动应用即可
